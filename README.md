# TransportFlow AI Frontend

Milestone 1 React frontend setup for the TransportFlow AI PRD.

## Scope

This project covers **Milestone 1: UI/UX Design & Setup** only.

Included:

- Vite + React frontend setup
- React Router app shell
- TransportFlow AI visual foundation
- Static overview dashboard mockup
- Static design-system reference page
- Reusable layout and UI components
- Plain CSS theme and responsive layout

Not included:

- Backend or API integration
- Authentication
- Database setup
- Real transportation workflows
- AI logic or optimization features
- Functional forms or persistent state

## Getting started

```bash
npm install
npm run dev
```

## Available scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Routes

- `/` — Static TransportFlow AI overview UI
- `/design-system` — Static UI foundation/reference page
- `*` — Static not-found page

## Milestone note

Future TMS modules such as loads, dispatch, fleet, customers, reports, settings, and AI-assisted insights are represented only as static placeholders or navigation labels. They are intentionally not functional in Milestone 1.
